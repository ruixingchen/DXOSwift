# DxOSwift

DxOSwift is an iOS client for DxOMark.com, unofficially


- 2017-10-31 - the first beta version is OK

#### Usage
clone or download the project, run `pod install` , then open DXOSwift.xcworkspace

#### Requirement

Xcode9, Swift4, iOS11
